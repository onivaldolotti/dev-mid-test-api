# Test Node Developer Mid Level

This is a sample Node.js project built with NestJS. It includes several endpoints to perform various calculations and operations.

### Prerequisites

- Node.js
- npm

### Getting Started

Clone this repository to your local machine:
``` bash
git clone https://github.com/onivaldolotti/dev-mid-test-api.git
cd dev-mid-test-api 
```

Install the dependencies:
``` bash
npm install 
```

Run the application in development mode:
``` bash
npm run start:dev 
```

The application will be running on http://localhost:3010.

### Swagger Documentation

The API documentation is available at http://localhost:3010/api/docs when the application is running. It provides detailed information about each endpoint, request schemas, and the available operations.

You can use the Swagger UI to interact with the endpoints and test the API.

### Testing
To run the unit tests, use the following command:

```bash
npm run test 
```
